**Create a new repository on the command line**
echo "# tomyris" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/cio-museums/tomyris.git
git push -u origin master

**Global config**
git config --global user.email "cio-museums@gmx.com"
git config --global user.name "cio-museums"
git config --list

**Local config**
git config user.email "cio-museums@gmx.com"
git config user.name "cio-museums"
git config --list

to set your account's default identity.
Omit --global to set the identity only in this repository.
